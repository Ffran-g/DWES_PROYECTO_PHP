# Nombre del Proyecto

Descripción breve:  
Este proyecto es un ejemplo de cómo usar **Bootstrap 5** en un proyecto web con **Visual Studio**. Incluye ejemplos de botones, tarjetas, barra de navegación y componentes responsivos.

---

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- [Bootstrap 5](https://getbootstrap.com/)
- Visual Studio Code

---

## Cómo ejecutar el proyecto
1. Clonar el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/nombre-proyecto.git
   ```
2. Abrir el proyecto en Visual Studio.
3. Abrir index.html en el navegador (o ejecutar desde Visual Studio si es un proyecto ASP.NET).
4. Verás la página con estilos y componentes de Bootstrap funcionando.
